# New Horizon

New Horizon is a collection of beginner Python practice scripts. The folder contains small exercises for learning Python basics, including input/output, functions, loops, conditionals, dictionaries, simple calculations, grading logic, voting eligibility checks, Fibonacci numbers, and pattern printing.

## Contents

- `calculator.py` - simple command-line calculator for arithmetic operations.
- `function.py` - examples of defining and calling functions.
- `fibonacci.py` - prints a short Fibonacci sequence.
- `for.py` and `while.py` - loop practice using `for` and `while`.
- `miif.py`, `Multiple_elif.py`, and `Multiple _elif1.py` - score grading examples using `if`, `elif`, and `else`.
- `voter.py` and `Election.py` - voting eligibility checks based on age.
- `dic.py` and `dic_example.py` - dictionary and nested dictionary examples.
- `Excercise.py` - loop and pattern-printing exercises.
- `sample.py` - assorted Python practice snippets.
- `DevOp.py` - generator practice with alternating number output.
- `documentation.txt` - short notes on Python topics to review.

## Requirements

- Python 3

No external packages are required.

## How to Run

Run any script from the project folder with:

```bash
python3 filename.py
```

Example:

```bash
python3 calculator.py
```

Some scripts ask for user input in the terminal.

## Purpose

This project is for practicing core Python programming concepts and building confidence with small command-line programs.
